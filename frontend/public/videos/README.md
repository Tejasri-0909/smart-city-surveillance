# Sample CCTV Video Files

This directory should contain sample surveillance footage for the camera feeds.

## Required Files:
- cam1.mp4 - City Center surveillance footage
- cam2.mp4 - Metro Station surveillance footage  
- cam3.mp4 - Airport Gate surveillance footage
- cam4.mp4 - Shopping Mall surveillance footage
- cam5.mp4 - Park Entrance surveillance footage
- cam6.mp4 - Highway Bridge surveillance footage

## Video Requirements:
- Format: MP4 (H.264)
- Duration: 10-30 seconds (will loop)
- Resolution: 640x480 or 1280x720
- Bitrate: Low to medium for smooth playback

## Sample Video Sources:
You can use any surveillance-style footage or create placeholder videos.
For testing, you can use any MP4 video files and rename them accordingly.

## Note:
If video files are not available, the system will fall back to simulated camera feeds with animated placeholders.
