# CCTV Video Files

This directory should contain realistic surveillance footage for each camera location.

## Required Files:
- `cam1.mp4` - City Center traffic footage
- `cam2.mp4` - Metro Station passenger footage  
- `cam3.mp4` - Airport Gate terminal footage
- `cam4.mp4` - Shopping Mall crowd footage
- `cam5.mp4` - Park Entrance walkway footage
- `cam6.mp4` - Highway Bridge traffic footage

## Video Requirements:
- Format: MP4
- Duration: 30-120 seconds (will loop continuously)
- Resolution: 720p or higher recommended
- File size: Under 50MB each for optimal loading
- Content: Realistic CCTV-style surveillance footage

## Suggested Sources (obtain proper licenses):
- Pexels.com (free with attribution)
- Pixabay.com (free with attribution)  
- Unsplash.com (free with attribution)
- Stock footage sites with proper licensing
